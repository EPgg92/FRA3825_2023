Les Balises HTML
  • Déterminées par un standard : impossible d'en inventer ou de les modifier
  • S'imbriquent (à l'exception des balises auto-fermantes) : pas de chevauchement (< p > < i > texte < /p > < /i >)

  • Peuvent/doivent avoir des attributs : pour baliser une image, il faut obligatoirement donner la source de l'image (< img src="lien" >)
